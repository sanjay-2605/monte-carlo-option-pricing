# Monte Carlo Option Pricing & Risk Simulation Engine

This project implements Monte Carlo simulation for stock price modeling, option pricing, and risk analysis.

## Features
- Stock price simulation
- European call option pricing
- Value at Risk (VaR)
- Volatility calculation
- Simulation plots
- Payoff distribution

## Technologies
Python, NumPy, Pandas, Matplotlib

## How to Run
pip install -r requirements.txt
python main.py# monte-carlo-option-pricing
